# video-check-tool
動画のテロップチェックツール

## 概要
動画のテロップを解析し、NGワードや誤字を検出するツールです。

## 使用技術
- Python
- Streamlit

## 主な機能
- 動画アップロード（mp4 / mov）
- エラー表示
- チェック実行
- 結果表示
- ダウンロード機能
